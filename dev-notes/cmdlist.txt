Event Commands
  !add-event e - adds the event to the list
  !remove-event - removes the event from the list
  !pick-event - picks a random event
  !events - displays events (used for first time setup)

Watchlist Commands
  !nominate s - adds show to candidate list
  !hiatus s - adds winner to hiatus section
  !banish s - adds winner to banished section
  !update s, # - updates the progress marker
  !remove s - scans entire JSON and removes item
  !finished s - adds show to finished section
